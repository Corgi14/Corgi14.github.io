关于Replugin

Replugin是 360 开发的占坑类插件化方案, 它让你的独立app(下称”插件”)可以以插件的形式运行被其他app(下称”宿主”)所使用, 支持本地和外置插件, 插件在宿主内部以apk包形式存在, 极易管理.

宿主三步集成Replugin

Github 宿主接入指南

  • 宿主根目录 build.gradle 添加host依赖
  • 宿主 app/build.gradle 添加依赖
  • 配制 Application 类

声明自定义Application

插件两步集成Replugin

Github 插件接入指南

  • 宿主根目录 build.gradle 添加plugin依赖
  • 插件 app/build.gradle 添加库依赖
