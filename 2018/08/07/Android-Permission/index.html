Android Permission(runtime)

###需求
一个简单的拨打电话的demo, 通过界面上的按钮来进行拨号. 初次进入App时需要检查&&开通权限, 如果用户没有开通, 则弹出提示框提示用户到设置界面自行开通.

###实现

  • 初始化
  • 检查权限
  • 重写权限注册的回调

###关于 ActivityCompat.shouldShowRequestPermissionRationale 方法

  • 这个方法的返回值分以下几种情况
权限对话框选择情况 方法返回值
第一次打开App时 false
上次弹出权限点击了禁止 true
上次选择禁止并勾选”下次不再询问” false
